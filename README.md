# Gigeo_Bill_System
## About  
Gigeo Bill System is a frontend-heavy application built with the Vite + Tailwind CSS framework stack.  
It is designed to provide a modern, responsive UI for billing operations, with mock authentication, item listings, charts, and localStorage persistence for improved UX.

## Tech Stack  
- Vite  
- React (or Vanilla JS)  
- Tailwind CSS  
- Recharts (for charts)  
- LocalStorage for recent searches / user sessions  
- Mock JSON data for items  

## Features  
- Simple login system (mock)  
- Responsive dashboard layout with sidebar + topbar  
- Item listing with search, filter, sort  
- Item detail modal with extended information  
- Chart view showing items by category  
- Theme toggle (light/dark) with preference stored  
- Recent searches stored locally  

## Set Up
npm install
npm run dev
Open  http://localhost:5173/
